## Unsplash Experiment

## Steps

`npm i`

`npm run start`

Make sure you create a .env variable and add REACT_APP_ACCESS_KEY

### Tech used

-   React
-   Styled-components

### Features

-   Search for Images
-   Filter based on different filters
-   Load images infinite on bottom scroll

### Filters Used

-   SORT BY
    -   Relevance
    -   Newest
-   COLOR
    -   Any
    -   Black and White
-   ORIENTATION
    -   Any
    -   LandScape
    -   Portrait
    -   Square
-   CONTENT FILTER
    -   Low
    -   High
